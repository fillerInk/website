Ee Javascriptil vayassu kandethunnathu enganeyaanennu kaanano ningalkku?
Kaanam

		const calcAge = (dateString) => {

		const birthday = new Date(dateString).getTime();

		const msInYear = 365.25*24*60*60*1000;

		return parseInt((Date.now() - birthday) / msInYear);

		}
Aadyam ingane function declare cheyyanam.
Ennitingane vilikanam calcAge(“12 Jan 1999”) appo 21 ennu kittum!

2 Reactions

Pinne oru kaaryam. Aa dateString ee roopathilokke adikkanam:

  • 2020 02 01 (yyyy mm dd)
  • 01 Feb 2020 (dd mmm yyyy)
  • 1264962600000 (milliseconds since 1970)

Muzhuvan rekha kaanano? Ithaana rekha: ECMA 262 Specification
